# cola-lista-pilas
En las siguientes tareas se le agregan distintas funcionalidades a los diferentes codigo segun los requisitos solicitados por el profesor encargado de la materia.


# 1. Lista

## Funcionalidades implementadas
- Agregar tareas
- Marcar tarea como completada
- Eliminar tarea por nombre
- Contar tareas registradas

## Objetivo
Practicar el manejo de listas y operaciones básicas sobre colecciones de datos.

---

# 2. Pilas

## Funcionalidades implementadas
- Navegación hacia atrás
- Función `Forward()` utilizando otra pila
- Límite máximo de historial de 10 elementos

## Objetivo
Comprender el funcionamiento de las pilas mediante la simulación de un historial de navegación.

---

# 3. Cola

## Funcionalidades implementadas
- Atención por prioridad (VIP primero)
- Función `Peek()` para ver el siguiente elemento sin eliminarlo
- Gestión de cola de espera

## Objetivo
Aplicar el uso de colas y prioridades en sistemas de atención.
